Our Bodies Are Art: An interactive examination of evolving female beauty standards through machine learning, computer vision, and classical art

Isabel Lee, Advisor: Julie Dorsey

Abstract:
When we examine the diverse types of bodies found in art throughout time, we better understand our past standards of beauty: who and what we have exalted and immortalized. By extension, we discover that there is no ‘ideal body’, as each artist and time period bring their own unique and subjective eye. This project explores this idea through an interactive web application that matches a user’s body to a number of similar figures found in classical art. First, a diverse art database of around 50 works was curated from sources like ArtStor, WikiArt, and various Museum databases like Tate and the Guggenheim. Then, the TensorFlow.js library BodyPix was applied to each image, using machine learning and computer vision to accurately segment bodies. Data was collected and standardized; then a Vantage Point tree data structure was implemented to efficiently find the k nearest neighbors for any input data. Finally, a web app was built to capture a user’s body, with the BodyPix library segmenting the image with user correction. Users are then able to explore the five closest matches from the art database and see how they compare to their own body. This proof-of-concept application lays the foundation for many promising extensions; future work may entail expanding the database and creating more curated matches by also taking into account race, age, and gender. Overall, this project allows users to engage in classical art while also reflecting on their own ideas of beauty and body image through an interactive and compelling application.
